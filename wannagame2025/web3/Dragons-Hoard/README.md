# Dragon's Hoard

Can you drain the dragon's hoard?

## Files
- `program/`: The smart contract source code.
- `server/`: The challenge server code.
- `challenge/`: Docker setup for local testing.

## Goal
Exploit the program to drain the vault. The server listens on port 31337.
You need to write a Solana program to solve this.

## Local Setup
```bash
cd challenge
docker-compose up --build
```
